# AWS Visitor Counter 🚀

This is a serverless web application that tracks website visits using AWS.

## 🌐 Live Demo

(http://visitor-counter-site-2026.s3-website-us-east-1.amazonaws.com/)

## 🧰 Technologies Used

* Amazon S3 (Static Website Hosting)
* AWS Lambda
* Amazon DynamoDB
* Amazon API Gateway

## ⚙️ How It Works

1. User visits the website hosted on S3
2. JavaScript calls API Gateway
3. API Gateway triggers Lambda
4. Lambda updates DynamoDB
5. Updated visitor count is returned and displayed

## 📌 Features

* Real-time visitor counter
* Fully serverless architecture
* Low-cost and scalable
